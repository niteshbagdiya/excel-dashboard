# excel-dashboard
Interactive Excel dashboard analyzing pizza sales with KPIs, pivot charts, slicers, and timeline filters. Tracks revenue, orders, best/worst sellers, and trends by day, hour, size, and category. Built using data cleaning, Pivot Tables, and visualization to deliver clear, actionable business insights.
